# 深圳理工大学智能与行为交叉研究中心申请材料准备包

本文件夹用于准备深圳理工大学人工智能研究院“智能与行为交叉研究中心”相关岗位申请材料。

## 文件说明

- `研究计划_草稿.md`：面向该中心方向定制的研究计划草稿。
- `岗位信息与材料清单.md`：招聘信息摘要、需要补齐的申请材料、邮件投递提示。
- `Google_Scholar_WOS_Profile_Placeholder.md`：学术档案链接占位文件，后续替换为 Google Scholar 或 Web of Science 信息。
- `materials/01_Guangrui_Fan_CV_Chinese.pdf`：中文 CV。
- `materials/01_Guangrui_Fan_CV_Chinese_latest.pdf`：中文 CV 备份副本。
- `materials/01_CV_Chinese_Source.tex`：中文 CV LaTeX 源文件副本。
- `materials/02_Research_Talk_Slides.pdf`：现有科研/面试汇报 PDF。
- `materials/02_Research_Talk_Slides.pptx`：现有科研/面试汇报 PPTX。
- `materials/02_Research_Talk_Slides_Source.md`：现有科研/面试汇报 Markdown 源文件。
- `materials/selected_publication_*.md`：与研究计划最相关的代表性论文条目。

## 建议下一步

1. 确认拟申请岗位名称：特聘副教授、博士后、专职研究人员或其他。
2. 将 `研究计划_草稿.md` 中的岗位名称、时间规划、预期成果数量按实际申请档位微调。
3. 补充 Google Scholar / Web of Science 链接或截图。
4. 确认推荐信人选与邮箱。
5. 按招聘公告要求整理邮件主题：`樊光瑞+应聘智能与行为研究中心+应聘岗位`。
